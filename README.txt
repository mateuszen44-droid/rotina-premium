# Rotina Premium Pro - Mateus (Desktop)

## ✅ O que é isso?
Um app desktop (Windows/Mac/Linux) feito em Electron com visual premium (dark + vermelho), rotina, progresso e salvamento automático.

## 🚀 Como rodar
1) Instale o Node.js (LTS)
2) Abra o terminal dentro da pasta do projeto
3) Rode:

npm install
npm start

## 📦 Como gerar instalador (.exe)
Depois de instalar dependências:

npm run dist

Vai criar a pasta **release/** com o instalador do Windows (NSIS).

## 📌 Observação importante
O ZIP vem com o **código pronto**. As dependências (node_modules) são baixadas pelo `npm install`.
Isso é normal em qualquer projeto Node/Electron.
